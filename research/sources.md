# 文献表 / Synthesis Matrix

> 项目主题：夹心层世代的时间困境，以及工作、育儿、养老等正当任务如何挤压个体的自主时间。  
> 证据强度说明：高 = 同行评审实证研究/权威机构报告；中 = 经典理论、综述、政策文件、可靠新闻；低 = 预印本、二手页面、需进一步核实的材料。
